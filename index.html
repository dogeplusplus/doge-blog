Catalan - Classification of Cat Sounds by Emotion

Have you ever wanted to understand what a cat is trying to say? Well now we are one step closer to realising this technology. cat-alan is an app hosted with Streamlit that takes uploaded audio/video of cat sounds, and tries to categorise them into 1 of 10 emotions. The model is based off the M5 architecture trained on roughly 3000 labelled cat audio clips. A link to the deployment can be found here.

Ball Balancer - Training Unity ML Agents using PPO

This is a demo of training an agent using Unity ML-Agents. This shows how to interact with environments provided by Unity, and uses a PyTorch implementation of PPO from OpenAI's spinning up library, instead of using mlagents-learn directly. There is some parallelisation of environments using MPI which allows for faster training.

Neural Style - Image Stylization by Mixing Representations

neural-style makes use of a pre-trained model from TensorFlow Hub. By uploading a content image and a style one can create funky art. This is currently deployed as a HuggingFace space and can be found here.

Seafood Distillation - CNN Model Distillation for Seafood Image Classification

PyTorch demonstration of how to perform model distillation using the KL-divergence loss term. An example of student teacher networks.

Termivision - TV in the Command Line

Rust video player that prints directly to the terminal. Using the ansi_term create one can print coloured pixels. termivision shows how to use the OpenCV rust bindings to parse image/video data and transform them into coloured text.
